# Requisitos do Sistema

## Funcionais
1. Adicionar uma tarefa.
2. Listar todas as tarefas.
3. Atualizar descrição de uma tarefa.
4. Excluir uma tarefa.
5. (Mudança de escopo) Marcar uma tarefa como concluída.

## Não Funcionais
1. O sistema deve ser implementado em JavaScript (Node.js).
2. O código deve ser testável e conter testes automatizados.
3. O repositório deve ter CI configurado no GitHub Actions.
4. O projeto deve seguir boas práticas de versionamento e Kanban.
